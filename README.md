# Strategic EV Charging Infrastructure Analysis

## Project Overview

This project analyzes Electric Vehicle (EV) adoption and charging
infrastructure availability across India using Power BI. It identifies
growth patterns, infrastructure shortages, and future demand projections
to support data-driven decision-making.

## Key Features

-   Interactive Power BI dashboard
-   EV sales trend analysis (2022--2024)
-   Market share analysis by manufacturer
-   State-wise EV penetration analysis
-   Charging infrastructure gap analysis
-   CAGR and growth rate calculations
-   2030 EV demand projections

## Tools & Technologies

-   Power BI
-   Excel
-   Data Cleaning
-   Data Visualization
-   Data Analysis
-   Dashboard Development

## Business Value

This project demonstrates the ability to: - Analyze large datasets -
Identify trends and patterns - Build interactive dashboards - Support
strategic decision-making - Communicate insights effectively
